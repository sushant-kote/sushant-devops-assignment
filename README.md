# DevOps Assignment

## Overview
This project demonstrates deployment of a production-like Node.js application on AWS EC2 using Docker and GitHub Actions.

## Technologies Used

- AWS EC2
- Docker
- GitHub Actions
- Amazon S3
- CloudWatch
- IAM
- Ubuntu Linux
- Node.js
- Git

## Features

- Dockerized Node.js application
- Automated CI/CD pipeline
- Secure SSH deployment
- CloudWatch monitoring
- S3 backup
- Load testing using k6
- Security Groups and IAM
